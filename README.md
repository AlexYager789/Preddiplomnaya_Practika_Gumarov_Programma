## Описание проекта

Система предназначена для автоматического анализа обращений в call-центре университета с использованием методов машинного обучения (NLP).

### Основные возможности
- Классификация обращений
- Анализ тональности и срочности
- Кластеризация системных проблем
- Формирование аналитических отчётов
- Многоуровневая система доступа
